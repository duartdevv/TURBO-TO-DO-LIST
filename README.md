### TO-DO Turbo

# Funcionalidades

- Adicionar tarefas;
- Marcar como conclúida(✔);
- Remover tarefas individualmente;
- Limpar TODAS as tarefas de uma só vez;
- Bloqueio de input de tarefa vazia;
- Contador automático;
- Totalmente responsivo 
- Botão de download da lista:
    Faz o download de todos os itens da lista em um arquivo .txt, sendo os que estão marcados como CONCLÚIDOS serão destacados na lista. Exemplo > Tarefas 1, 3, 8, 9 marcads como conclúidas. No arquivo de texto ficará com um símbolo ✔ de "Concluído". Exemplo:

- [ ✔ ] Tarefa 1
  
- Caso o usuário tente fazer o download da lista VAZIA, irá ser exibido um pop-up dizendo que não é possivel realizar o download de uma lista vazia;

### Layout

- Tema escuro moderno, com botões destacados e animações suaves. Interface limpa e fácil de usar;

### Técnologias usadas

- HTML5;
- CSS3;
- JavaScript(DOM);

### Responsividade

- O layout se adapta automáticamente a telas pequenas, tornando a experiência boa tanto no PC, quanto em dispositivos menores;

### Como usar

1. Digite uma tarefa no campo de texto;
2. Clique em Adicionar ou aperte "Enter";
3. Use '✔' para marcar como conclúida - Ou use 'X' para REMOVER;
5. Clique em 'Limpar tudo' para apagar tudo da lista;
