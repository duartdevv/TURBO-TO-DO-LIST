### TO-DO Turbo

# Funcionalidades

- Adicionar tarefas;
- Marcar como conclúida(✔);
- Remover tarefas individualmente;
- Limpar TODAS as tarefas de uma só vez;
- Bloqueio de input de tarefa vazia;
- Contador automático;
- Totalmente responsivo 
- Botão de download da lista:
    Faz o download de todos os itens da lista em um arquivo .txt, sendo os que estão marcados como CONCLÚIDOS serão destacados na lista. Exemplo > Tarefas 1, 3, 8, 9 marcads como conclúidas. No arquivo de texto ficará assim:
             [✔] Tarefa 1
             [ ] Tarefa 2
             [✔] Tarefa 3
             [ ] Tarefa 4
             [ ] Tarefa 5
             [ ] Tarefa 6
             [ ] Tarefa 7
             [✔] Tarefa 8
             [✔] Tarefa 9
             [ ] Tarefa 10
    Caso o usuário tente fazer o download da lista VAZIA, irá ser exibido um pop-up dizendo o mesmo;

### Responsividade

- O layout se adapta automáticamente a telas pequenas, tornando a experiência boa tanto no PC, quanto em dispositivos menores;

### Como usar

1. Digite uma tarefa no campo de texto;
2. Clique em Adicionar ou aperte "Enter";
3. Use '✔' para marcar como conclúida - Ou use 'X' para REMOVER;
5. Clique em 'Limpar tudo' para apagar tudo da lista;
